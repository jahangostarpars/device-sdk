## Test Connection

command: `test:connection`

> no params

**Example Request**:

```json
{
  "device": {
    "host": "192.168.20.20",
    "port": 8080,
    "type": "RadenT41",
    "password": "123"
  },
  "command": "test:connection"
}
```

**Example Response**:

```http
HTTP/1.1 200 Ok

{
  message: "Connection Successful"
}
```
